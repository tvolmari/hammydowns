WordPress Marketplace is a free full-featured Shopping Cart / eCommerce Plugin with everything you need on an easy UI, to build eShop or marketplace.

== Description ==

The WordPress Marketplace plugin is a full-featured WordPress Shopping Cart / eCommerce solution that is extremely easy to install and even easier to maintain. Transform your WordPress website into a full featured online store just in few click.
Whatever you need related to eCommerce / Shopping Cart System, like: building a online shop, full featured marketplace, deal site, coupon site or membership site WordPress Marketplace is a perfect plugin for your need. As well as enterprise-level quality and features, its user friendly and hassle free UI will give you a
exclusive and comfortable experience.

1. Upload `wpmarketplace` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
